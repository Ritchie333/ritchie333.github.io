9ACA: Print a group of graphics
The buffer contains the number of entries in the first byte, plus that number of the following:
Offset Field
0 Attribute
1 Y co-ordinate
2 X co-ordinate
3 UDG ID (offset from CD0C)
Input
IX Pointer to graphic buffer
9ACA LD B,(IX+$00)
9ACD INC IX
9ACF LD A,(IX+$00)
9AD2 LD D,(IX+$01)
9AD5 LD E,(IX+$02)
9AD8 LD C,(IX+$03)
9ADB PUSH BC
9ADC CALL $9A90
9ADF POP BC
9AE0 INC IX
9AE2 INC IX
9AE4 INC IX
9AE6 INC IX
9AE8 DJNZ $9ACF
9AEA RET
