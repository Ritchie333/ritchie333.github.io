C700: Vector jump table
This table is used to work around limitations with Interrupt Mode 2 on the Spectrum. When this mode is enabled, the I register contains the top 8 bytes of an address. Because the bottom 8 bytes are undefined, the actual jump will be randomly anywhere in this buffer. As the two bytes are the same, the jump is guaranteed to be 5B5B. This is a well-known hack on the ZX Spectrum.
